Flappy Bird Game

A Python implementation of the classic Flappy Bird game, built using the Pygame library. This project is a fun and interactive game where the player controls a bird trying to navigate through pipes without crashing.

Features

Smooth and responsive gameplay

Retro-style graphics and animations

Score tracking

Realistic gravity and collision mechanics

Installation

Prerequisites

Ensure you have Python 3.x installed on your system. Additionally, you will need the Pygame library.

Install Pygame using pip:

pip install pygame

Clone the Repository


How to Play

Run the game:


Press the Spacebar to make the bird fly.

Navigate the bird through the pipes without hitting them or falling.

Each successful pass through a pipe increases your score.
